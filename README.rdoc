== README

This is the website for St. Hughs' Ball, 2015. We're launching MT, 2014. Live site will be at https://sthughsball.com, currently, site is at http://beta.sthughsball.com

Built with Ruby on Rails (Devise, Sidekiq, Railsbricks), Postgresql, Redis (for Sidekiq), and Bootstrap. Uses nginx as a reverse-proxy, with Phusion Passenger on the backend. Integrates with Stripe and Mailchimp.
